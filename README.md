# UAE Electricity & Energy Consumption Dataset (2019–2024)

## Overview
This dataset contains electricity consumption data across selected emirates in the United Arab Emirates (UAE) from 2019 to 2024.

The data is categorized into:
- Residential Consumption
- Commercial Consumption
- Industrial Consumption
- Total Electricity Consumption

All electricity values are measured in Gigawatt-hours (GWh).

---

# Dataset Structure

| Column Name | Description |
|---|---|
| Year | Reporting year |
| Emirate | Name of the UAE emirate |
| Residential_GWh | Residential electricity consumption in GWh |
| Commercial_GWh | Commercial electricity consumption in GWh |
| Industrial_GWh | Industrial electricity consumption in GWh |
| Total_GWh | Total electricity consumption in GWh |

---

# Covered Emirates

- Abu Dhabi
- Dubai
- Sharjah
- Ajman
- Ras Al Khaimah

2019 – 2024
